Đã lưu
Cách dùng — 3 bước:
  1. Sửa nội dung bên dưới (thêm/xóa hãng, xe, đổi tên, hạn mức, ảnh…). Mọi thay đổi tự lưu vào máy bạn để xem thử ngay trên “Mở trang web”.
  2. Xem ổn rồi, bấm “Tải file dữ liệu (.json)” để tải về file f88-xe-data.json.
  3. Đưa file f88-xe-data.json lên web (cùng thư mục với trang chính) là khách sẽ thấy nội dung mới.
